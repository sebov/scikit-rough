# Rough Sets Knowledge Base

This knowledge base documents the theoretical foundations of rough set theory. It consists of
definitions, propositions, and examples covering core concepts such as decision tables,
indiscernibility, approximations, decision reducts, decision rules, consistency, classification, and
evaluation.

The goal is to provide a solid theoretical reference for the rough set-based methods
implemented in the project. All notation follows the conventions defined in
`notation_and_symbols.md`.
